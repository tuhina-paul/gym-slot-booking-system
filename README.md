# Gym Slot Booking System

A full-stack Gym Slot Booking web application built using **Flask, SQLite, and JavaScript**.

# Features
- User Registration & Login
- Secure Password Hashing (bcrypt)
- JWT Authentication
- Slot Booking & Cancellation
- Prevents Double Booking
- User-specific Bookings
- RESTful APIs
- Frontend + Backend Integration

# Tech Stack
- Backend: Python, Flask
- Database: SQLite
- Frontend: HTML, CSS, JavaScript
- Authentication: JWT
- Security: bcrypt
